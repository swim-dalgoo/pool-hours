# 수영장 자유수영 시간표

여러 수영장의 자유수영 운영시간을 한 화면에서 보는 웹앱입니다.
지금 갈 수 있는 곳이 맨 위에 뜨고, 자기가 다니는 수영장을 직접 추가·수정할 수 있습니다.
서버 없이 브라우저만으로 동작하며, 편집한 내용은 각자 기기에 저장됩니다.

## 바로 쓰기

https://아이디.github.io/pool-hours/

휴대폰에서는 크롬으로 연 뒤 메뉴 → "홈 화면에 추가" 하면 앱처럼 쓸 수 있습니다.

## 기능

- 개장 중 → 곧 개장 → 종료 → 휴관 순 자동 정렬
- 05~21시 타임라인에 현재 시각 표시
- 요일별 시간표, "이 요일 복사" 로 반복 요일 한 번에 입력
- 메모에 "강습" 을 넣으면 자유수영과 구분해 표시
- 공휴일 탭 + 공휴일 날짜 등록 시 당일 자동 전환
- 백업·복원 텍스트로 다른 기기·다른 사람과 시간표 공유

## 참고

시설 사정으로 운영시간이 바뀔 수 있으니 중요한 날은 해당 수영장에 확인하세요.
